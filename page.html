Cài Đặt Môi Trường Lập Trình (macOS)

1. IDE (Visual Studio Code)

Cài đặt: Tải và cài đặt trực tiếp từ Visual Studio Code
Cấu hình hệ thống: Sau khi cài, mở bảng lệnh (Cmd + Shift + P), tìm và chọn: 
Install 'code' command in PATH

2. Lập trình C/C++

Trong Terminal, chạy lần lượt các lệnh:

xcode-select --install
clang --version

Trong VS Code, cài đặt Extension: C/C++

3. Lập trình C#

3.1. Cài đặt Homebrew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
echo $SHELL
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
brew --version
which brew

4. Lập trình .NET

4.1. Cài đặt .NET SDK

brew install --cask dotnet-sdk
dotnet --version
Lưu ý: Nếu báo lỗi command not found, chạy tiếp các lệnh sau: 
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zprofile
source ~/.zprofile
which dotnet

Trong VS Code, cài đặt Extension: C# Dev Kit

5. Lập trình Python

brew install python
echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zprofile
source ~/.zprofile
python3 --version
which python3
brew install python@3.10
python3.10 --version
which python3.10

5.1. Cấu hình trong VS Code

  • Cài đặt Extension: Python
  • Nhấn Cmd + Shift + P, copy và dán lệnh sau vào: 
    Python: Select Interpreter
  • Sau đó chọn đường dẫn (interpreter) có chứa homebrew.

6. Lập trình Mobile (Flutter & Dart)

6.1. Cài đặt Flutter

brew install --cask flutter
echo 'export PATH="$PATH:/opt/homebrew/Caskroom/flutter/latest/flutter/bin"' >> ~/.zprofile
source ~/.zprofile

6.2. Cấu hình Xcode & Công cụ

Cài Xcode trong AppStore sau đó cài Emulator. Sau đó làm các bước sau:

Bước 1: Kiểm tra đường dẫn

xcode-select -p

Bước 2: Xử lý kết quả

  • Nếu kết quả là: /Applications/Xcode.app/Contents/Developer
    • Đúng, không cần thực hiện thêm bước nào.
  • Nếu kết quả là: /Library/Developer/CommandLineTools
    • Đang sử dụng Command Line Tools. Hãy chuyển sang Xcode bằng lệnh: 
      sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer

Bước 3: Khởi tạo Xcode

sudo xcodebuild -runFirstLaunch

6.3. Cài đặt CocoaPods

brew install cocoapods
pod --version
which pod

6.4. Cài đặt Android Studio

Tải tại: Android Studio

Cài SDK: Android SDK Platform-Tools + Android SDK Command-line Tools + Android Emulator

Sau đó chạy:

echo 'export ANDROID_HOME=$HOME/Library/Android/sdk' >> ~/.zprofile
echo 'export PATH="$PATH:$ANDROID_HOME/emulator"' >> ~/.zprofile
echo 'export PATH="$PATH:$ANDROID_HOME/platform-tools"' >> ~/.zprofile
source ~/.zprofile

Kết quả kiểm tra:

echo $ANDROID_HOME
-> /Users/baonguyenn/Library/Android/sdk
which adb
-> .../platform-tools/adb
which emulator
-> .../emulator/emulator

6.5. Kiểm tra flutter doctor

flutter doctor --android-licenses
flutter doctor -v

8. Cơ sở dữ liệu & Container (Docker + MySQL)

8.1. Docker

Tải trực tiếp từ web và cài đặt: Docker Desktop

Kiểm tra cài đặt:

docker --version

8.2. MySQL qua Docker

Tạo và chạy container MySQL:

docker run --name baonguyenn-mysql -e MYSQL_ROOT_PASSWORD=Bao@23022005 -p 3306:3306 -d mysql:latest

Dừng container:

docker stop baonguyenn-mysql

8.3. MySQL Workbench

Tải và cài đặt phiên bản dành cho Mac từ trang chủ MySQL: MySQL Workbench

8.4. VS Code Extensions

Cài đặt các Extension sau:

  • SQLTools
  • SQLTools MySQL/MariaDB/TiDB
